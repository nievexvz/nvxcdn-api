API Info

Base URL: https://nievexsviz.my.id
API Key: nvxc
Version: v1

Overview

NVXCloud adalah API untuk upload file (CDN) dan URL Shortener yang sederhana dan ringan.

CDN Features

  • Upload file hingga 25MB
  • Auto-generate short filename (6 karakter)
  • Support semua jenis file

Shortener Features

  • Custom ID support
  • Auto-generate ID (5 karakter)
  • Click tracking

Authentication

API Key Required

Semua request ke API endpoint membutuhkan API Key yang valid melalui header x-api-key

Example Header 
x-api-key: nvxc

Error Response

{
  "success": false,
  "message": "API Key required atau tidak valid"
}

CDN API

POST /api/v1/cdn
File Upload

Upload file dan file akan disimpan dengan nama pendek (6 karakter) dan bisa diakses via URL pendek.

cURL Example

Terminal 
curl -X POST https://nievexsviz.my.id/api/v1/cdn \
  -H "x-api-key: nvxc" \
  -F "file=@gambar.jpg"

Node.js Example

Request Body

Form Data:
file: [File Binary]

Success Response

{
  "success": true,
  "message": "File berhasil diupload",
  "data": {
    "id": "aB3x9y",
    "filename": "aB3x9y.jpg",
    "originalName": "gambar.jpg",
    "size": 1024000,
    "mimeType": "image/jpeg",
    "url": "https://nievexsviz.my.id/aB3x9y.jpg"
  }
}

Shortener API

POST /api/v1/short
URL Shortening

Buat short URL dari URL panjang.

cURL Example

Terminal 
curl -X POST https://nievexsviz.my.id/api/v1/short \
  -H "x-api-key: nvxc" \
  -H "Content-Type: application/json" \
  -d '{"url":"https://google.com"}'

Node.js Example

Request Body

{
  "url": "https://example.com",
  "customId": "optional-custom-id"
}

Success Response

{
  "success": true,
  "message": "Short URL berhasil dibuat",
  "data": {
    "id": "abc12",
    "original_url": "https://google.com",
    "short_url": "https://nievexsviz.my.id/r/abc12",
    "createdAt": "2024-01-15T10:30:00.000Z"
  }
}

Quick Examples

Quick Upload 

# Create test file
echo "Hello World" > test.txt

# Upload
curl -X POST https://nievexsviz.my.id/api/v1/cdn \
  -H "x-api-key: nvxc" \
  -F "file=@test.txt"

Quick Shorten 

# Create short URL
curl -X POST https://nievexsviz.my.id/api/v1/short \
  -H "x-api-key: nvxc" \
  -H "Content-Type: application/json" \
  -d '{"url":"https://github.com"}'

Access Examples

Access Uploaded File

# Via browser
https://nievexsviz.my.id/aB3x9y.jpg

# Via curl
curl https://nievexsviz.my.id/aB3x9y.jpg

Access Short URL

# Via browser (auto redirect)
https://nievexsviz.my.id/r/abc12

# Via curl (follow redirect)
curl -L https://nievexsviz.my.id/r/abc12
